# JavaScript Testing Practice with Jest

The goal of this project was for us to practice using Jest to test the following JavaScript functions:

- capitalize;
- reverse;
- calculator add, subtract, multiply and divide;
- Ceasar Cipher;
- analyze array;

You can find the original description in [The Odin Project](https://www.theodinproject.com/courses/javascript/lessons/testing-practice)

## Built With

- JavaScript
- npm
- Jest
- Babel

## Getting Started

Download or clone the project [here](https://github.com/bruna-genz/testing-practice.git), then enter in the project's directory from the terminal and run the following comands:
- `npm i`
- `npm run test`
